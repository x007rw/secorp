## 表示方法

### 方法1：直接ファイルを開く（最簡）

エクスプローラーで各HTMLをダブルクリック → ブラウザで開く

※画像は `../public/images/` を参照しているため、フォルダ階層を維持してください。

### 方法2：Astroの開発サーバー経由

```bash
cd C:\dev\secorp
npm run dev
```

その後ブラウザで：
- http://localhost:4321/design-proposals/01-current.html

※ Astro 開発サーバー経由なら、画像パスが確実に解決されます。

## デザイン比較ポイント

各案で**同じコンテンツ・同じ素材**を使い、**ビジュアルだけが異なる**設計にします。

| 比較軸 | 評価項目 |
|---|---|
| 第一印象 | 何の会社か即座に伝わるか |
| 信頼感 | 老舗らしさ・実直さが出ているか |
| 業界適合性 | BtoB建設・土木業界の慣習に合っているか |
| 運用しやすさ | 文章追加時のレイアウト崩れリスク |
| モバイル表示 | スマホでも読みやすいか |
