# デザイン提案 — 4 案

社長室長より「老舗の蕎麦屋」のイメージをいただき、それに対する解釈を含む 4 つのデザイン方向を作成しました。

## ファイル構成

```
design-proposals/
├── 01-current.html             モダン B2B（現行Astro版の単一HTML移植）
├── 02-shop-traditional.html    老舗・蕎麦屋風（古風・実直）
├── 03-conservative.html        保守的法人系（地方銀行風）
├── 04-minimal.html             ミニマル工業系（製図感）
└── README.md
```

## 各案の方向性

### 01-current.html — モダン B2B
- **配色**：スレート紺 ＋ アンバー金
- **印象**：洗練された海外メーカー風、テック寄り
- **長所**：現代的で見栄え良い、CTAが目立つ
- **短所**：「老舗の蕎麦屋」とは反対方向、業界慣習から浮く可能性
- **想定企業例**：川崎重工コーポレートサイト

### 02-shop-traditional.html — 老舗・蕎麦屋風 ★社長室長のご要望に最も近い
- **配色**：藍色 ＋ 朱色 ＋ 和紙ベージュ ＋ 古金
- **タイポ**：見出しに Noto Serif JP（明朝体）
- **印象**：創業の重み、実直、信頼感、時代に流されない
- **特徴**：
  - 「創業 一九六七年」を冒頭から強調
  - 漢数字表記（一九七二年〜、八千万円 等）で老舗感
  - のれん風ヘッダー、罫線・囲み枠を多用
  - 朱印風スタンプで「日本総代理店」を強調
  - ヒーローは画像+グラデで控えめ、文字中心
- **長所**：信頼感・歴史性が即伝わる、業界顧客に受け入れられやすい
- **短所**：若い世代・海外顧客には堅すぎる印象

### 03-conservative.html — 保守的法人系
- **配色**：紺白基調 ＋ 控えめな金アクセント
- **タイポ**：Noto Sans JP のみ（標準ゴシック）
- **印象**：地方銀行・建設業協会・公的機関的
- **特徴**：
  - 上部の「お知らせバー」（住所・TEL常時表示）
  - 公式書類風の表組み多用
  - サイドラベル＋本文の二段組レイアウト
  - 数値クイックバー（創業年・資本金・事業歴・代表者）
  - ヒーローは小さめ、すぐ情報セクションへ
- **長所**：BtoB の調達担当者が情報を素早く取得できる、整然
- **短所**：個性が薄い、印象に残りにくい

### 04-minimal.html — ミニマル工業系
- **配色**：黒・白・グレーのみ（無彩色）
- **タイポ**：細字 Noto Sans JP ＋ JetBrains Mono（等幅）
- **印象**：建築設計事務所・製図事務所・海外スタジオ風
- **特徴**：
  - 番号付きセクション（/ 001、/ 002…）で図面感
  - 製品コード（PRD-001 等）の付与
  - 写真はモノクロ寄り
  - 等幅フォントによる仕様書感
  - 英語ベース＋日本語補足
- **長所**：海外顧客に強い、グローバル展開意識のある会社印象
- **短所**：日本の伝統的な業界顧客には冷たく感じられる可能性
